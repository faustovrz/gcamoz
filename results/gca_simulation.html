[!NOTE] This notebook is dedicated strictly to the genetic simulation of a 7 × 7 maize factorial cross (Line × Tester) evaluated in two environments. The objective is to establish a biologically realistic dataset for General Combining Ability (GCA) and Specific Combining Ability (SCA) calculations, and save it in a clean CSV format.

1 Mating and Environmental Design

Our simulation is modeled exactly on Stelio’s PhD experimental setup: * Breeding Scheme: Factorial Mating Design (North Carolina Design II / Line × Tester). * Females (Lines): 7 Low-Phosphorus tolerant CIMMYT inbred lines (CML364, CML366, CML434, CML435, CML439, CML440, CML532). * Males (Testers): 7 elite local Mozambican lines (Local1 to Local7). * Crosses: All \(7 \times 7 = 49\) unique \(F_1\) hybrid combinations. (No selfs, no reciprocals). * Experimental Design: Randomized Complete Block Design (RCBD) with 3 replications (blocks) per environment. * Environments (\(E\)): 2 environments representing phosphorus treatments (env1 = Normal Phosphorus, env2 = Low Phosphorus Stress). * Response Trait: Grain Yield (tons/ha).

2.1 Parameter Definitions

We define the genetic and environmental parameters that will determine the phenotypic yield: 1. Base Mean (\(\mu\)): 8.2 tons/ha. 2. Environment Effects (\(E\)): Normal P (env1) adds \(+1.1\) t/ha; Low P (env2) reduces yield by \(-1.3\) t/ha. 3. Female GCA Effects (\(g_f\)): CIMMYT lines carry alleles for phosphorus use efficiency (PUE). 4. Male GCA Effects (\(g_m\)): Local lines reflect general agroecological adaptation. 5. SCA Effects (\(s_{fm}\)): Specific hybrid combinations show positive heterosis. 6. GCA × Environment Interaction (\(G \times E\)): In the low-P environment (env2), the PUE tolerance of superior CIMMYT lines (CML364 and CML366) is expressed even more strongly (enhancing their GCA by \(+30\%\)). 7. SCA × Environment Interaction (\(S \times E\)): Hybrid-specific adaptation to environmental stress. 8. Residual Plot Error (\(\epsilon\)): Normal variation among plots (\(\sigma_{\epsilon} = 0.35\) t/ha).

Code 
# Parental lines
females <- c("CML364", "CML366", "CML434", "CML435", "CML439", "CML440", "CML532")
males <- paste0("Local", 1:7)

# GCA effects
gca_female <- c(CML364=1.2, CML366=0.8, CML434=0.4, CML435=0.0, CML439=-0.3, CML440=-0.7, CML532=-1.1)
gca_male <- c(Local1=0.9, Local2=0.5, Local3=0.1, Local4=-0.2, Local5=-0.5, Local6=-0.8, Local7=-1.2)

# Specific Combining Ability (SCA) matrix
sca_matrix <- matrix(0, nrow=7, ncol=7, dimnames=list(females, males))
sca_matrix["CML364", "Local4"] <- 0.5   # Specific positive combining ability
sca_matrix["CML435", "Local1"] <- 0.4
sca_matrix["CML366", "Local6"] <- -0.4  # Specific negative incompatibility

3 Dataset Generation

We combine these components to simulate grain yields for all \(2 \text{ envs} \times 3 \text{ blocks} \times 49 \text{ crosses} = 294\) plot observations.

Code 
# 1. Create complete grid of observations
sim_data <- expand.grid(
  env = c("env1", "env2"),
  block = c("B1", "B2", "B3"),
  female = females,
  male = males,
  stringsAsFactors = FALSE
)

# 2. Add Cross ID
sim_data <- sim_data %>%
  mutate(cross = paste(female, male, sep = "_"))

# 3. Simulate Grain Yield (t/ha) based on our linear model
sim_data$yield <- apply(sim_data, 1, function(row) {
  env <- row["env"]
  blk <- row["block"]
  fem <- row["female"]
  mal <- row["male"]
  crs <- row["cross"]
  
  # Overall mean
  mu <- 8.2
  
  # Fixed environmental effect
  e_eff <- if(env == "env1") 1.1 else -1.3
  
  # Block nested in Environment (random block effect)
  set.seed(which(females == fem)*10 + which(males == mal) + which(c("env1", "env2") == env)*100 + which(c("B1", "B2", "B3") == blk)*1000)
  b_eff <- rnorm(1, mean = 0, sd = 0.18)
  
  # Parental General Combining Abilities
  g_fem <- gca_female[fem]
  g_mal <- gca_male[mal]
  
  # Specific Combining Ability
  s_val <- sca_matrix[fem, mal]
  
  # GCA x Environment Interaction
  # In env2 (low-P stress), the superior PUE donor GCAs are amplified by 30%
  ge_fem <- if(env == "env2") g_fem * 0.3 else -g_fem * 0.3
  ge_mal <- if(env == "env2") g_mal * 0.2 else -g_mal * 0.2
  
  # SCA x Environment Interaction (hybrid specific stress adaptation)
  se_val <- rnorm(1, mean = 0, sd = 0.10)
  
  # Residual Plot Error
  err <- rnorm(1, mean = 0, sd = 0.35)
  
  # Combine to get phenotypic yield
  y <- mu + e_eff + b_eff + g_fem + g_mal + s_val + ge_fem + ge_mal + se_val + err
  return(round(y, 3))
})
