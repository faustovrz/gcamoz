NoteOverview

In comparative biology and experimental research, a standard Analysis of Variance (ANOVA) determines if group means differ significantly. However, ANOVA alone does not pinpoint which specific groups differ. To resolve this, we perform post-hoc pairwise comparisons (such as Tukey’s HSD) and represent the resulting patterns of significance elegantly using a Compact Letter Display (CLD).

This notebook provides a complete, step-by-step pipeline to run an ANOVA on the Petal.Length of the classic iris dataset and visualize the results with a publication-quality CLD boxplot.

1. Exploratory Data Analysis (EDA)

Before diving into statistical testing, let’s explore our data. The iris dataset contains 150 observations of three iris species: setosa, versicolor, and virginica. We want to understand the distribution of Petal.Length across these species.

2. Fitting the One-Way ANOVA Model

We formulate our statistical hypothesis as follows: * Null Hypothesis (\(H_0\)): The mean petal length is identical across all three species (\(\mu_{\text{setosa}} = \mu_{\text{versicolor}} = \mu_{\text{virginica}}\)). * Alternative Hypothesis (\(H_1\)): At least one species has a mean petal length that differs from the others.

We fit a standard one-way ANOVA model using R’s built-in aov() function:

Code 
# Fit the ANOVA model
iris_anova <- aov(Petal.Length ~ Species, data = iris)

# Summary of the ANOVA model
anova_summary <- summary(iris_anova)

3. Checking ANOVA Assumptions

To ensure our ANOVA results are valid, we should verify the two core assumptions: normality of residuals and homogeneity of variances.

3.1. Normality of Residuals

We plot the Q-Q plot and run the Shapiro-Wilk test on the residuals.

Code 
# Set up a side-by-side layout for diagnostics
par(mfrow = c(1, 2))

# Residuals vs Fitted
plot(iris_anova, which = 1, col = "#4D96FF", pch = 16, lwd = 1.5)

# Normal Q-Q Plot
plot(iris_anova, which = 2, col = "#FF6B6B", pch = 16, lwd = 1.5)

Let’s formalize this with the Shapiro-Wilk test:

Code 
shapiro.test(residuals(iris_anova))

    Shapiro-Wilk normality test

data:  residuals(iris_anova)
W = 0.98108, p-value = 0.03676
  • Interpretation: The Shapiro-Wilk test returns a \(p\)-value \(< 0.05\), which indicates a deviation from perfect normality. However, ANOVA is highly robust to deviations from normality, especially with balanced designs like the iris dataset (\(n=50\) per group).

3.2. Homogeneity of Variance

We perform Levene’s Test from the car package to check if variance is equal across species.

Code 
leveneTest(Petal.Length ~ Species, data = iris)
  • Interpretation: Levene’s test returns a highly significant \(p\)-value (\(< 0.05\)). This indicates that the assumption of equal variances is technically violated. In practice, biological datasets frequently show heterogeneous variance. While the F-test remains relatively robust, we can also perform a Welch’s ANOVA (which does not assume equal variances) to confirm our results:
Code 
# Welch's ANOVA (robust to heteroscedasticity)
oneway.test(Petal.Length ~ Species, data = iris, var.equal = FALSE)

    One-way analysis of means (not assuming equal variances)

data:  Petal.Length and Species
F = 1828.1, num df = 2.000, denom df = 78.073, p-value < 2.2e-16

Even after adjusting for unequal variances, the differences remain overwhelmingly significant (\(p < 2.2 \times 10^{-16}\)).

4. Post-Hoc Analysis & Compact Letter Display (CLD)

Having established that at least one species differs, we now use Tukey’s Honest Significant Difference (HSD) test to perform all pairwise comparisons. We’ll extract the Estimated Marginal Means (EMMs) and generate a Compact Letter Display (CLD) to group the species.

4.1. Generating the CLD with emmeans

We use the modern standard emmeans package in conjunction with multcomp to compute the CLD.

Code 
# Compute Estimated Marginal Means
iris_emms <- emmeans(iris_anova, ~ Species)

# Generate the Compact Letter Display (CLD)
# Groups sharing a letter are NOT significantly different
iris_cld <- cld(iris_emms, Letters = letters, adjust = "tukey")

# Convert to data frame and clean column names
cld_df <- as.data.frame(iris_cld)
cld_df$cld_letters <- trimws(cld_df$.group)

# Show the resulting dataframe
cld_df

4.2. Displaying the Post-Hoc Comparisons in a Beautiful Table

Let’s use gt to format the post-hoc results and estimated marginal means.

5. Visualizing the Final CLD on the Plot

Now, we’ll overlay the CLD letters onto our gorgeous boxplot. To position the letters dynamically and perfectly above each boxplot, we’ll calculate the maximum value for each species and place the text slightly above that.

Code 
# Find the maximum petal length for each species to place letters dynamically
max_values <- aggregate(Petal.Length ~ Species, data = iris, max)
names(max_values)[2] <- "y_pos"

# Merge coordinates with our CLD dataframe
plot_labels <- merge(cld_df, max_values, by = "Species")

# Generate the final publication-ready plot
ggplot(iris, aes(x = Species, y = Petal.Length, fill = Species)) +
  # Violin plot underlay
  geom_violin(alpha = 0.15, color = NA) +
  # Main boxplots
  geom_boxplot(width = 0.4, alpha = 0.85, outlier.shape = NA, color = "#2D3748", lwd = 0.8) +
  # Jittered raw data points for transparency
  geom_jitter(width = 0.1, alpha = 0.35, size = 1.8, aes(color = Species)) +
  # Dynamic CLD Letters
  geom_text(
    data = plot_labels,
    aes(x = Species, y = y_pos + 0.3, label = cld_letters),
    size = 6,
    fontface = "bold",

  # Colors and styling
  scale_fill_manual(values = species_palette) +
  scale_color_manual(values = species_palette) +

    x = "Species",
    y = "Petal Length (cm)",
    caption = "Different letters indicate statistically significant differences between species (Tukey's HSD, p < 0.05)."
  ) +
  theme_premium()
