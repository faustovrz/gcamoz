1 Theoretical Foundation & Experimental Setup

Before performing calculations in R, we must define the genetic breeding scheme, the field experimental design, and the mathematical phenotype model.

1.1 Breeding Scheme (Line × Tester)

Stelio’s research focuses on the genetic architecture of low-phosphorus use efficiency (PUE) in maize. The mating design is a Factorial Mating Design (often referred to as a Line × Tester / North Carolina Design II). * Female Parents (Lines - PUE Donors from CIMMYT): 7 low-P tolerant inbred lines: \[\text{CML364, CML366, CML434, CML435, CML439, CML440, and CML532}\] * Male Parents (Testers - Local elite lines from Mozambique): 7 well-adapted but susceptible lines (labeled Local1 to Local7). * Hybrid Families (\(F_1\)): All possible crosses between the 7 females and 7 males are made, yielding a total of 49 unique \(F_1\) hybrid combinations. * Genetic Advantage: Unlike a diallel, a factorial mating design utilizes two distinct, mutually exclusive parental sets. This prevents selfing or reciprocal cross complications.

1.2 Experimental Design

The 49 \(F_1\) hybrids are evaluated under a Randomized Complete Block Design (RCBD) with 3 replications (blocks) per environment. * Environments (\(E\)): The trials are conducted across 2 environments representing phosphorus treatment levels: 1. env1: Normal/High Phosphorus conditions. 2. env2: Low Phosphorus Stress conditions. * Trait Evaluated: Grain Yield (tons/ha). * Total Plots: \(2 \text{ environments} \times 3 \text{ replications} \times 49 \text{ hybrid entries} = 294\) plot evaluations.

1.3 Phenotype Model (Mathematical Equation)

To analyze the phenotypic yield data, we model the value of a plot containing hybrid \(pq\) (cross of female parent \(p\) and male parent \(q\)) in block \(j\) of environment \(i\) as:

\[y_{ijpq} = \mu + E_i + B_{j(i)} + g_{F_p} + g_{M_q} + s_{pq} + (Eg_{F})_{ip} + (Eg_{M})_{iq} + (Es)_{ipq} + \epsilon_{ijpq}\]

Where: * \(y_{ijpq}\) is the grain yield observed in the plot. * \(\mu\) is the overall mean grain yield of the population. * \(E_i\) is the fixed effect of the \(i\)-th environment (phosphorus treatment). * \(B_{j(i)}\) is the random effect of the \(j\)-th block nested within the \(i\)-th environment, assumed \(B_{j(i)} \sim \text{NID}(0, \sigma^2_{B(E)})\). * \(g_{F_p}\) is the random General Combining Ability (GCA) effect of the \(p\)-th female parent, assumed \(g_{F_p} \sim \text{NID}(0, \sigma^2_{GCA\_F})\). This represents the additive breeding value transmitted by the CIMMYT donor. * \(g_{M_q}\) is the random GCA effect of the \(q\)-th male parent, assumed \(g_{M_q} \sim \text{NID}(0, \sigma^2_{GCA\_M})\). This represents the additive breeding value transmitted by the local tester. * \(s_{pq}\) is the random Specific Combining Ability (SCA) effect of the cross between female \(p\) and male \(q\), assumed \(s_{pq} \sim \text{NID}(0, \sigma^2_{SCA})\). This represents dominance deviations (heterosis/hybrid vigor). * \((Eg_{F})_{ip}\) and \((Eg_{M})_{iq}\) are the random interactions of the parental GCA effects with the environment, modeling how additive breeding values respond to low-phosphorus stress. * \((Es)_{ipq}\) is the random interaction of the SCA effect with the environment, representing specific hybrid adaptability to stress. * \(\epsilon_{ijpq}\) is the residual experimental plot error, assumed \(\epsilon_{ijpq} \sim \text{NID}(0, \sigma^2_{\epsilon})\).

[!IMPORTANT] Because Stelio’s design uses distinct female and male lines (a factorial mating scheme), the parental effects are mutually exclusive. Unlike a diallel, we do not need to overlay design matrices or use overlay functions in mixed models. This significantly simplifies our mixed model code in R!

4 Fixed-Effects Calculations: Factorial Combining Ability

In a balanced factorial cross, GCA and SCA can be calculated arithmetically from family means. This provides a transparent foundation before using mixed models.

4.1 Arithmetic Formulas

Let \(\bar{Y}_{p.}\) be the mean yield of all crosses involving female parent \(p\). Let \(\bar{Y}_{.q}\) be the mean yield of all crosses involving male parent \(q\). Let \(\bar{Y}_{pq}\) be the mean yield of the hybrid cross between female \(p\) and male \(q\). Let \(\bar{Y}_{..}\) be the overall grand mean of all crosses in the factorial grid.

  • Grand Mean: \[\bar{Y}_{..} = \frac{1}{f \cdot m} \sum_{p} \sum_{q} \bar{Y}_{pq}\]
  • GCA of Female Parent \(p\) (\(g_{F_p}\)): \[g_{F_p} = \bar{Y}_{p.} - \bar{Y}_{..}\]
  • GCA of Male Parent \(q\) (\(g_{M_q}\)): \[g_{M_q} = \bar{Y}_{.q} - \bar{Y}_{..}\]
  • SCA of Cross \(pq\) (\(s_{pq}\)): \[s_{pq} = \bar{Y}_{pq} - g_{F_p} - g_{M_q} - \bar{Y}_{..} = \bar{Y}_{pq} - \bar{Y}_{p.} - \bar{Y}_{.q} + \bar{Y}_{..}\]

4.2 Direct R Arithmetic

We calculate these effects arithmetically from the entire dataset.

Code 
# 1. Calculate family means
family_means <- maize_data %>%
  group_by(female, male) %>%
  summarize(mean_yield = mean(yield), .groups = 'drop')

# 2. Reshape to a 7x7 matrix (7 females in rows, 7 males in columns)
Y_matrix <- matrix(0, nrow = 7, ncol = 7, dimnames = list(levels(maize_data$female), levels(maize_data$male)))
for(i in 1:nrow(family_means)) {
  Y_matrix[as.character(family_means$female[i]), as.character(family_means$male[i])] <- family_means$mean_yield[i]
}

# 3. Grand mean
mu_hat <- mean(Y_matrix)

# 4. Female and Male means
female_means <- rowMeans(Y_matrix)
male_means <- colMeans(Y_matrix)

# 5. GCA effects
gca_female_fixed <- female_means - mu_hat
gca_male_fixed <- male_means - mu_hat

# 6. SCA effects
sca_fixed <- matrix(0, nrow = 7, ncol = 7, dimnames = list(levels(maize_data$female), levels(maize_data$male)))
for(p in rownames(sca_fixed)) {
  for(q in colnames(sca_fixed)) {
    sca_fixed[p, q] <- Y_matrix[p, q] - gca_female_fixed[p] - gca_male_fixed[q] - mu_hat
  }
}

5 Mixed-Effects Calculations: Factorial Variance Component Analysis

In modern breeding, combining ability is analyzed using Linear Mixed Models (LMMs). Parents are treated as random samples from a reference population.

We use the R package sommer to fit the mixed model. Because female and male lines are distinct, no matrix overlays are needed. We analyze the Low Phosphorus Stress environment (env2) where selecting tolerant lines is critical.

5.1 Fitting the Mixed Model

The random terms in our model are: female (Female GCA), male (Male GCA), cross (SCA), and block (RCBD replication).

Code 
# Filter for env2 (Low Phosphorus Stress)
data_env2 <- maize_data %>%
  filter(env == "env2") %>%
  mutate(
    female = as.factor(female),
    male = as.factor(male),
    cross = as.factor(cross),
    block = as.factor(block)
  )

# Fit Mixed Model in sommer
# yield = fixed intercept + random female GCA + random male GCA + random SCA + random block
fit_factorial <- mmer(
  fixed = yield ~ 1,
  random = ~ female + male + cross + block,
  data = data_env2,
  verbose = FALSE
)

# Extract and display variance components

6 Breeding Values (BV) and Genotypic Values (GV)

For hybrid breeding selection, we must calculate the Expected Breeding Value (BV) and the Genotypic Value (GV) of the crosses: 1. Breeding Value (\(BV\)): The sum of the parent general combining abilities. \(BV\) represents the purely additive value of the cross. \[BV_{\text{Cross } PQ} = g_{F_p} + g_{M_q}\] 2. Genotypic Value (\(GV\)): The total genetic value of the cross. \(GV\) equals the Breeding Value plus the Specific Combining Ability (dominance deviation). \(GV\) represents the commercial potential of the hybrid. \[GV_{\text{Cross } PQ} = BV_{\text{Cross } PQ} + SCA_{PQ} = g_{F_p} + g_{M_q} + s_{pq}\]

We calculate these for all 49 crosses under Low Phosphorus stress (env2) using our mixed model results.

7 Heritability Estimation from Factorial Variance Components

We use the estimated genetic and environmental variance components from our mixed model (fit_factorial) to calculate the narrow-sense (\(h^2\)) and broad-sense (\(H^2\)) heritabilities.

7.1 Heritability Formulas for Factorial Designs

According to Chapter 5 of Isik et al. (2017): * Additive Variance (\(\sigma^2_A\)): Since female and male parental sets are distinct, additive variance is estimated as: \[\sigma^2_A = 2 \cdot (\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M})\] * Dominance Variance (\(\sigma^2_D\)): Estimated as: \[\sigma^2_D = 4 \cdot \sigma^2_{SCA}\] * Phenotypic Variance among Individuals (\(\sigma^2_{\text{phenotypic}}\)): \[\sigma^2_{\text{phenotypic}} = \sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \sigma^2_{\text{residual}}\] * Narrow-Sense Individual Heritability (\(h^2_i\)): \[h^2_i = \frac{\sigma^2_A}{\sigma^2_{\text{phenotypic}}} = \frac{2 \cdot (\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M})}{\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \sigma^2_{\text{residual}}}\] * Broad-Sense Individual Heritability (\(H^2_{bs}\)): \[H^2_{bs} = \frac{\sigma^2_A + \sigma^2_D}{\sigma^2_{\text{phenotypic}}} = \frac{2 \cdot (\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M}) + 4 \cdot \sigma^2_{SCA}}{\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \sigma^2_{\text{residual}}}\] * Full-Sib Family Heritability (\(h^2_{FS}\)): Represents the accuracy of selection among whole families based on plot means: \[\sigma^2_{FS\_mean} = \sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \frac{\sigma^2_{\text{residual}}}{r}\] \[h^2_{FS} = \frac{\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M}}{\sigma^2_{FS\_mean}}\] where \(r = 3\) replicates.

7.2 Direct R Calculations

We execute these calculations in R using the variance components retrieved from fit_factorial.

Code 
# Extract individual variance components from sommer output
var_gca_f <- var_comps["female:yield:yield", "VarComp"]
var_gca_m <- var_comps["male:yield:yield", "VarComp"]
var_sca <- var_comps["cross:yield:yield", "VarComp"]
var_res <- var_comps["units:yield:yield", "VarComp"]

# Calculate genetic variances
var_A <- 2 * (var_gca_f + var_gca_m)
var_D <- 4 * var_sca

# Individual Phenotypic Variance
var_phen_ind <- var_gca_f + var_gca_m + var_sca + var_res

# Calculate Individual Heritabilities
h2_individual_ns <- var_A / var_phen_ind
H2_individual_bs <- (var_A + var_D) / var_phen_ind

# Full-Sib Family mean Phenotypic Variance (r = 3 replicates)
r_rep <- 3
var_phen_fs <- var_gca_f + var_gca_m + var_sca + (var_res / r_rep)

# Calculate Family Heritability
h2_family_fs <- (var_gca_f + var_gca_m) / var_phen_fs

# Print results
heritability_results <- tibble(
  Parameter = c(
    "Additive Genetic Variance (Va)",

    "Individual Phenotypic Variance (Vp_ind)",
    "Narrow-Sense Individual Heritability (h2_i)", 
    "Broad-Sense Individual Heritability (H2_bs)",

## Experimental Design
The 49 $F_1$ hybrids are evaluated under a **Randomized Complete Block Design (RCBD)** with **3 replications (blocks)** per environment.
*   **Environments ($E$):** The trials are conducted across **2 environments** representing phosphorus treatment levels:
    1.  **env1:** Normal/High Phosphorus conditions.
    2.  **env2:** Low Phosphorus Stress conditions.
*   **Trait Evaluated:** Grain Yield (tons/ha).
*   **Total Plots:** $2 \text{ environments} \times 3 \text{ replications} \times 49 \text{ hybrid entries} = 294$ plot evaluations.

***

## Phenotype Model (Mathematical Equation)
To analyze the phenotypic yield data, we model the value of a plot containing hybrid $pq$ (cross of female parent $p$ and male parent $q$) in block $j$ of environment $i$ as:

$$y_{ijpq} = \mu + E_i + B_{j(i)} + g_{F_p} + g_{M_q} + s_{pq} + (Eg_{F})_{ip} + (Eg_{M})_{iq} + (Es)_{ipq} + \epsilon_{ijpq}$$

Where:
*   $y_{ijpq}$ is the grain yield observed in the plot.
*   $\mu$ is the overall mean grain yield of the population.
*   $E_i$ is the fixed effect of the $i$-th environment (phosphorus treatment).
*   $B_{j(i)}$ is the random effect of the $j$-th block nested within the $i$-th environment, assumed $B_{j(i)} \sim \text{NID}(0, \sigma^2_{B(E)})$.
*   $g_{F_p}$ is the random General Combining Ability (GCA) effect of the $p$-th female parent, assumed $g_{F_p} \sim \text{NID}(0, \sigma^2_{GCA\_F})$. This represents the additive breeding value transmitted by the CIMMYT donor.
*   $g_{M_q}$ is the random GCA effect of the $q$-th male parent, assumed $g_{M_q} \sim \text{NID}(0, \sigma^2_{GCA\_M})$. This represents the additive breeding value transmitted by the local tester.
*   $s_{pq}$ is the random Specific Combining Ability (SCA) effect of the cross between female $p$ and male $q$, assumed $s_{pq} \sim \text{NID}(0, \sigma^2_{SCA})$. This represents dominance deviations (heterosis/hybrid vigor).
*   $(Eg_{F})_{ip}$ and $(Eg_{M})_{iq}$ are the random interactions of the parental GCA effects with the environment, modeling how additive breeding values respond to low-phosphorus stress.
*   $(Es)_{ipq}$ is the random interaction of the SCA effect with the environment, representing specific hybrid adaptability to stress.
*   $\epsilon_{ijpq}$ is the residual experimental plot error, assumed $\epsilon_{ijpq} \sim \text{NID}(0, \sigma^2_{\epsilon})$.

> [!IMPORTANT]
> Because Stelio's design uses distinct female and male lines (a factorial mating scheme), the parental effects are mutually exclusive. Unlike a diallel, **we do not need to overlay design matrices or use overlay functions** in mixed models. This significantly simplifies our mixed model code in R!

# Fixed-Effects Calculations: Factorial Combining Ability

In a balanced factorial cross, GCA and SCA can be calculated arithmetically from family means. This provides a transparent foundation before using mixed models.

## Arithmetic Formulas
Let $\bar{Y}_{p.}$ be the mean yield of all crosses involving female parent $p$.
Let $\bar{Y}_{.q}$ be the mean yield of all crosses involving male parent $q$.
Let $\bar{Y}_{pq}$ be the mean yield of the hybrid cross between female $p$ and male $q$.
Let $\bar{Y}_{..}$ be the overall grand mean of all crosses in the factorial grid.

*   **Grand Mean:**
    $$\bar{Y}_{..} = \frac{1}{f \cdot m} \sum_{p} \sum_{q} \bar{Y}_{pq}$$
*   **GCA of Female Parent $p$ ($g_{F_p}$):**
    $$g_{F_p} = \bar{Y}_{p.} - \bar{Y}_{..}$$
*   **GCA of Male Parent $q$ ($g_{M_q}$):**
    $$g_{M_q} = \bar{Y}_{.q} - \bar{Y}_{..}$$
*   **SCA of Cross $pq$ ($s_{pq}$):**
    $$s_{pq} = \bar{Y}_{pq} - g_{F_p} - g_{M_q} - \bar{Y}_{..} = \bar{Y}_{pq} - \bar{Y}_{p.} - \bar{Y}_{.q} + \bar{Y}_{..}$$

## Direct R Arithmetic

We calculate these effects arithmetically from the entire dataset.

```{r}
#| label: arithmetic-calculations

# 1. Calculate family means
family_means <- maize_data %>%
  group_by(female, male) %>%
  summarize(mean_yield = mean(yield), .groups = 'drop')

# 2. Reshape to a 7x7 matrix (7 females in rows, 7 males in columns)
Y_matrix <- matrix(0, nrow = 7, ncol = 7, dimnames = list(levels(maize_data$female), levels(maize_data$male)))
for(i in 1:nrow(family_means)) {
  Y_matrix[as.character(family_means$female[i]), as.character(family_means$male[i])] <- family_means$mean_yield[i]
}

# 3. Grand mean
mu_hat <- mean(Y_matrix)

# 4. Female and Male means
female_means <- rowMeans(Y_matrix)
male_means <- colMeans(Y_matrix)

# 5. GCA effects
gca_female_fixed <- female_means - mu_hat
gca_male_fixed <- male_means - mu_hat

# 6. SCA effects
sca_fixed <- matrix(0, nrow = 7, ncol = 7, dimnames = list(levels(maize_data$female), levels(maize_data$male)))
for(p in rownames(sca_fixed)) {
  for(q in colnames(sca_fixed)) {
    sca_fixed[p, q] <- Y_matrix[p, q] - gca_female_fixed[p] - gca_male_fixed[q] - mu_hat
  }
}

# Print Female GCA Effects
gca_female_df <- tibble(
  Female_Parent = names(gca_female_fixed),
  Mean_Yield = round(female_means, 3),
  Estimated_GCA = round(gca_female_fixed, 4)
)

kable(gca_female_df, caption = "Estimated GCA Effects for Female Parents (CIMMYT Lines)") %>%
  kable_styling(bootstrap_options = c("striped", "hover", "condensed"))

# Print Male GCA Effects
```

# Breeding Values (BV) and Genotypic Values (GV)

For hybrid breeding selection, we must calculate the **Expected Breeding Value (BV)** and the **Genotypic Value (GV)** of the crosses:
1.  **Breeding Value ($BV$):** The sum of the parent general combining abilities. $BV$ represents the purely additive value of the cross.
    $$BV_{\text{Cross } PQ} = g_{F_p} + g_{M_q}$$
2.  **Genotypic Value ($GV$):** The total genetic value of the cross. $GV$ equals the Breeding Value plus the Specific Combining Ability (dominance deviation). $GV$ represents the commercial potential of the hybrid.
    $$GV_{\text{Cross } PQ} = BV_{\text{Cross } PQ} + SCA_{PQ} = g_{F_p} + g_{M_q} + s_{pq}$$

We calculate these for all 49 crosses under Low Phosphorus stress (`env2`) using our mixed model results.


# Heritability Estimation from Factorial Variance Components

We use the estimated genetic and environmental variance components from our mixed model (`fit_factorial`) to calculate the narrow-sense ($h^2$) and broad-sense ($H^2$) heritabilities.

## Heritability Formulas for Factorial Designs
According to Chapter 5 of Isik et al. (2017):
*   **Additive Variance ($\sigma^2_A$):** Since female and male parental sets are distinct, additive variance is estimated as:
    $$\sigma^2_A = 2 \cdot (\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M})$$
*   **Dominance Variance ($\sigma^2_D$):** Estimated as:
    $$\sigma^2_D = 4 \cdot \sigma^2_{SCA}$$
*   **Phenotypic Variance among Individuals ($\sigma^2_{\text{phenotypic}}$):**
    $$\sigma^2_{\text{phenotypic}} = \sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \sigma^2_{\text{residual}}$$
*   **Narrow-Sense Individual Heritability ($h^2_i$):**
    $$h^2_i = \frac{\sigma^2_A}{\sigma^2_{\text{phenotypic}}} = \frac{2 \cdot (\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M})}{\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \sigma^2_{\text{residual}}}$$
*   **Broad-Sense Individual Heritability ($H^2_{bs}$):**
    $$H^2_{bs} = \frac{\sigma^2_A + \sigma^2_D}{\sigma^2_{\text{phenotypic}}} = \frac{2 \cdot (\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M}) + 4 \cdot \sigma^2_{SCA}}{\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \sigma^2_{\text{residual}}}$$
*   **Full-Sib Family Heritability ($h^2_{FS}$):** Represents the accuracy of selection among whole families based on plot means:
    $$\sigma^2_{FS\_mean} = \sigma^2_{GCA\_F} + \sigma^2_{GCA\_M} + \sigma^2_{SCA} + \frac{\sigma^2_{\text{residual}}}{r}$$
    $$h^2_{FS} = \frac{\sigma^2_{GCA\_F} + \sigma^2_{GCA\_M}}{\sigma^2_{FS\_mean}}$$
    where $r = 3$ replicates.

## Direct R Calculations
We execute these calculations in R using the variance components retrieved from `fit_factorial`.

```{r}
#| label: heritability-factorial

# Extract individual variance components from sommer output
var_gca_f <- var_comps["female:yield:yield", "VarComp"]
var_gca_m <- var_comps["male:yield:yield", "VarComp"]
var_sca <- var_comps["cross:yield:yield", "VarComp"]
var_res <- var_comps["units:yield:yield", "VarComp"]

# Calculate genetic variances
var_A <- 2 * (var_gca_f + var_gca_m)
var_D <- 4 * var_sca

# Individual Phenotypic Variance
var_phen_ind <- var_gca_f + var_gca_m + var_sca + var_res

# Calculate Individual Heritabilities
h2_individual_ns <- var_A / var_phen_ind
H2_individual_bs <- (var_A + var_D) / var_phen_ind

# Full-Sib Family mean Phenotypic Variance (r = 3 replicates)
r_rep <- 3
var_phen_fs <- var_gca_f + var_gca_m + var_sca + (var_res / r_rep)

# Calculate Family Heritability
h2_family_fs <- (var_gca_f + var_gca_m) / var_phen_fs

# Print results
heritability_results <- tibble(
  Parameter = c(
    "Additive Genetic Variance (Va)", 
    "Dominance Genetic Variance (Vd)", 
    "Individual Phenotypic Variance (Vp_ind)",
    "Narrow-Sense Individual Heritability (h2_i)", 
    "Broad-Sense Individual Heritability (H2_bs)",
    "Narrow-Sense Full-Sib Family Heritability (h2_FS)"
  ),
  Value = round(c(var_A, var_D, var_phen_ind, h2_individual_ns, H2_individual_bs, h2_family_fs), 4)
)
```
